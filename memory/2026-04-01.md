## Discord Bot Project (Jane - Indonesian)

### Goal
Build automated Discord bot for livestock order system:
- Webhook integration from external livestock system
- Products posted to Discord via `/livestock` command
- Users click ORDER button → modal form (qty + notes)
- Generate ticket + QRIS payment code + total
- Orders stored in JSON

### Architecture
- **Tech:** Node.js + discord.js (migrated from Python)
- **Server:** Express webhook receiver
- **Storage:** JSON files (products.json, orders.json)
- **Config:** Environment variables via .env

### Progress
- ✅ Migrated from Python (discord.py + Flask) to Node.js (discord.js + Express)
- ✅ Created slash commands: `/addproduct`, `/addstock`, `/listproducts`, `/livestock`
- ✅ Product management (add/list/update stock) via JSON storage
- ✅ `/livestock` command posts all products to specified channel with ORDER buttons
- ✅ Order modal captures qty + notes
- ✅ Code pushed to GitHub: https://github.com/rzamlna/discordbot
- ✅ Cleaned workspace (deleted .md files except README)
- 🔄 **In Progress:** Payment integration (QRIS)

### Current Decision Point
- Evaluating QRIS payment provider:
  - **Midtrans** (professional, well-documented, higher fees)
  - **PaKasir** (local Indonesia, simpler, budget-friendly)
  - Jane leaning toward PaKasir but open to recommendation

### Key Files
- `bot.js` — Main Discord bot + Express server
- `package.json` — Dependencies
- `commands/` — Slash commands (addproduct, addstock, listproducts, livestock)
- `utils/orderModal.js` — Modal builder for order form
- `.env.example` — Config template
- `products.json` — Product storage
- `orders.json` — Order log

### Next Task
